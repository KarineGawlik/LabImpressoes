# Laboratório de Impressões

Website developed for Laboratório de Impressões, a printing and graphic services business.

🔗 Live project: https://lab-impressoes.vercel.app/

## Technologies

- React
- TypeScript
- HTML
- CSS
- Vercel

## Purpose

This project was created to present the business, its graphic services, contact information and online presence through a responsive website.

## Features

- Business presentation
- Services section
- Contact information
- Responsive layout
- Deployed application with Vercel
